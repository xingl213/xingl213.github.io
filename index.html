Recaller

Written in Java (GUI in Swing), this software helps people memorize by using actively recalling methods and flashcards. Whether you're trying to memorize German vocabulary, facts about World War II, or syntax of git commands -- this is the software that helps! As part of the learning goal of CSC207 Software Design, this software adheres fully to clean architecture and SOLID design principles. I'm happy to talk more about it with you if you're interested!

Pension Planning Tool

Mom asked me to help her with pension planning, and this tool was what I gave her. It started simple but in the end it made me very proud!
